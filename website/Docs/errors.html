Errors

Zekken reports syntax, reference, type, and runtime errors with line/column pointers.

  • Syntax errors: parser expectations and token mismatch.
  • Reference errors: missing variables/functions.
  • Type errors: declaration and operation mismatches.
  • Runtime errors: execution failures, such as missing files.
